Язык программирования PHP

Типы

Целые числа

Integer - это число из множества ? = {..., -2, -1, 0, 1, 2, ...}.

Строки

Строка - это набор символов, где символ - это то же самое, что и байт. Это значит, что PHP поддерживает ровно 256 различных символов, а также то, что в PHP нет встроенной поддержки Unicode

Массивы

Массив в PHP - это упорядоченное отображение, которое устанавливает соответствие между значением и ключом.

Управляющие конструкции языка PHP

if

Конструкция if является одной из наиболее важных во многих языках программирования, в том числе и PHP. Она предоставляет возможность условного выполнения фрагментов кода. Структура if реализована в PHP по аналогии с языком C:

else

Часто необходимо выполнить одно выражение, если определенное условие верно, и другое выражение, если условие не верно. Именно для этого else и используется. else расширяет оператор if, чтобы выполнить выражение, в лучае если условие в операторе if равно FALSE. for

Цикл for самый сложный цикл в PHP. Он ведет себя так же как его аналог в языке C.
